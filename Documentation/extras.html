This file will help you getting hold of some Theme Extras

Settings up Content Slider

<div class="fslider" data-speed="800" data-direction="vertical" data-pause="3000">

    <div class="flexslider">
    
    
        <div class="slider-wrap">
        
        
            <div class="slide">
            
                <img src="images/blog/1.jpg" alt="Post Gallery 1" />
            
            </div>
            
            
            ... and so on
        
        
        </div>
    
    
    </div>

</div>

Available Options:

  • data-animate - Animation Style, ie. slide or fade
  • data-easing - Easing Option, ie. swing or one of the easing methods from jQuery Easing
  • data-direction - Direction of Scrolling, ie. horizontal or vertical
  • data-slideshow - Disable/enable Auto Slideshow, ie. true or false
  • data-pause - Pause Time between Slides in milliseconds, ie. 4000 for 4 Seconds
  • data-speed - Speed of the Animation in milliseconds, ie. 400 for 0.4 seconds
  • data-video - Set true if you have video emded code in your slider

Background Images & Patterns

You can setup Background Images & Patterns Easily.

Setup Pattern

Add this in the html/css/colors.php

body {
    background-image: url("link-to-your-image.png");
}

Setup Background Image

Add this in your html File just after the opening <body> Tag.

<img src="your-background-image.jpg" alt="Background Image" id="imageBackground" />

Contact Form Email Processing

Just change your Email Address in the functions/sendemail.php and functions/footeremail.php file to start receiving Form Responses.

$toemail = 'youremail@yourwebsite.com';

Not receiving Form Emails.?

If you are not receiving Emails from your Forms then chances are that your Server Configuration doesn't support PHP mail() function. But you can use SMTP Authentication. Just find the following line of code:

$mail = new PHPMailer();

Add this after the above line:

$mail->IsSMTP();
$mail->Host = "mail.yourdomain.com";
$mail->SMTPDebug = 2;
$mail->SMTPAuth = true;
$mail->Host = "mail.yourdomain.com";
$mail->Port = 26;
$mail->Username = "yourname@yourdomain.com";
$mail->Password = "yourpassword";
