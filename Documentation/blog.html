Blog Preview

This theme supports 6 Blog Layouts and supports Images, Videos & Slider Galleries.

  • html/blog.html - Default Blog Layout
  • html/blog-alt.html - Alternate Blog Layout
  • html/blog-full.html - Full Width Blog Layout
  • html/blog-full-alt.html - Alternate Full Width Blog Layout
  • html/blog-small.html - Small Thumbnails Layout
  • html/blog-small-full.html - Full Width Small Thumbnails Layout

Info You can find all the Code Samples for the Sliders in the above files.
